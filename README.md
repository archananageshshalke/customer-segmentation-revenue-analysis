# 📊 Customer Segmentation & Revenue Analysis

## 📌 Project Overview
This project analyzes customer behavior and revenue patterns using *SQL and Power BI*.  
The goal is to identify high-value customers, understand sales trends, and generate actionable business insights for decision-making.

---

## 🛠 Tools & Technologies
- SQL (SQLite) – Data analysis  
- Power BI – Dashboard & visualization  
- Excel – Dataset preparation  

---

## 📂 Dataset
The dataset includes:
- seg_customers.csv  
- seg_orders.csv  
- seg_products.csv  
- seg_order_items.csv  

These tables are connected using keys such as *customer_id, order_id, and product_id*.

---

## 📊 Dashboard Preview
![Dashboard](customer_segmentation_dashboard.png)

---

## 📊 Analysis Performed (SQL)
- Customer order analysis  
- Revenue calculation using quantity and price  
- Monthly revenue trend analysis  
- Top customers by revenue  
- Top products by revenue  
- Customer segmentation (High, Medium, Low value)  
- Revenue contribution by customer segments  

---

## 📈 Power BI Dashboard Features
- KPI cards: Total Revenue, Total Orders, Total Customers, Average Order Value (AOV)  
- Monthly Revenue Trend (line chart)  
- Top 10 Customers by Revenue  
- Top 10 Products by Revenue  
- Interactive slicers (Date, Category)  

---

## 💡 Key Business Insights
- Top customers contribute a significant portion of total revenue, indicating revenue concentration among a few customers  
- Revenue shows monthly variation, highlighting seasonal trends and business fluctuations  
- Certain products dominate sales performance, suggesting strong product-market demand  
- Medium-value customers form a strong revenue base and offer growth potential  

---

## 📁 Project Structure
- seg_customers.csv → Customer dataset  
- seg_orders.csv → Orders dataset  
- seg_products.csv → Products dataset  
- seg_order_items.csv → Order items dataset  
- sales_analysis_queries.sql → SQL analysis queries  
- Customer_Segmentation_Revenue_Analysis.pbix → Power BI dashboard  
- customer_segmentation_dashboard.png → Dashboard screenshot  

---

## 🚀 How to Use
1. Open the .pbix file in Power BI  
2. Use slicers to filter data dynamically  
3. Explore insights through visuals  

---

## 🎯 Conclusion
This project demonstrates how *SQL and Power BI* can be combined to perform end-to-end data analysis and build interactive dashboards for business insights.
